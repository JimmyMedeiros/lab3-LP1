void Agencia::criar_conta ( )

Usando as classes implementadas, escreva um programa em C++ para simular uma agência bancária.

Uma agência bancária deve armazenar um conjunto de contas e permitir as seguintes operações básicas: criações de conta, exclusão de contas, saques (respeitando o saldo e o limite), depósitos, emissão de saldo e extrato, além de transferência entre contas.
